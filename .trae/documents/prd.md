## 1. 产品概述

**声影汇（SoundFlix）** — 一款集音乐与影视于一体的免费资源聚合平台，每日自动采集网络免费资源，提供在线浏览、试听/试看和下载功能，适配桌面端与移动端。

- 面向热爱音乐和影视的普通用户，解决免费资源分散、难以一站式获取的痛点
- 通过每日定时采集，确保内容新鲜度，打造"每日发现"的沉浸体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 游客 | 无需注册 | 浏览、搜索、试听/试看、下载 |
| 注册用户 | 邮箱注册 | 收藏、播放历史、个性化推荐 |

### 2.2 功能模块

1. **首页**：每日推荐轮播、热门音乐榜单、热门影视榜单、最新采集内容
2. **音乐专区**：音乐分类浏览、搜索、在线播放、下载
3. **影视专区**：影视分类浏览、搜索、详情查看、下载
4. **搜索页**：全局搜索、筛选与排序
5. **个人中心**：收藏管理、播放历史、设置

## 3. 核心流程

### 3.1 用户浏览与下载流程

用户打开应用 → 浏览首页推荐 → 进入音乐/影视专区 → 搜索或分类筛选 → 点击资源卡片查看详情 → 在线试听/试看 → 点击下载按钮 → 资源下载到本地

```mermaid

    "用户打开应用" --> "浏览首页推荐"
    "浏览首页推荐" --> "进入音乐专区"
    "浏览首页推荐" --> "进入影视专区"
    "进入音乐专区" --> "分类筛选/搜索"
    "进入影视专区" --> "分类筛选/搜索"
    "分类筛选/搜索" --> "点击资源卡片"
    "点击资源卡片" --> "查看详情"
    "查看详情" --> "在线试听/试看"
    "查看详情" --> "点击下载"
    "点击下载" --> "资源下载到本地"
```

### 3.2 每日数据采集流程

定时任务触发 → 调用免费API获取最新资源 → 数据清洗与去重 → 存入数据库 → 更新前端展示

```mermaid
    "数据清洗与去重" --> "存入数据库"
    "存入数据库" --> "更新前端展示"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色系（#0A0E17 深空黑）+ 金色点缀（#D4A853 琥珀金）
- **辅助色**：#1A1F2E（深蓝灰）、#2A3040（中灰蓝）、#E8E8E8（浅灰文字）
- **按钮风格**：圆角微3D效果，hover时发光渐变
- **字体**：标题使用 Playfair Display（优雅衬线体），正文使用 DM Sans（现代无衬线体）
- **布局风格**：侧边导航 + 内容区卡片流，移动端底部Tab导航
- **图标风格**：Lucide线性图标，金色高亮
- **动效**：页面切换淡入、卡片hover缩放+阴影、播放器波形动画、粒子背景

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 每日推荐轮播 | 全宽渐变背景、大图+文字叠加、自动轮播指示器 |
| 首页 | 热门榜单 | 横向滚动卡片、金色排名徽章、hover放大效果 |
| 首页 | 最新采集 | 瀑布流网格、时间标签、渐入动画 |
| 音乐专区 | 播放器 | 底部固定栏、唱片旋转动画、波形可视化、进度条 |
| 影视专区 | 详情页 | 毛玻璃背景海报、评分圆环、标签云 |
| 搜索页 | 搜索栏 | 大号搜索框、实时下拉建议、分类Tab |
| 个人中心 | 整体 | 暗色卡片布局、金色分割线、渐变头像框 |

### 4.3 响应式设计

- 桌面优先设计，断点：768px（平板）、480px（手机）
- 桌面端：侧边导航 + 多列网格
- 平板端：折叠侧边栏 + 两列网格
- 手机端：底部Tab导航 + 单列布局 + 触摸优化（大按钮、滑动手势）

### 4.4 3D场景指引

- 首页背景：微粒子星空效果，缓慢飘动，营造沉浸感
- 音乐播放器：CSS 3D唱片旋转效果
- 影视详情：海报视差滚动效果
